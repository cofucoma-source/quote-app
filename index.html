📘 Claudeプロジェクトのセットアップ手順を見る
  1. Claude.ai にログイン →「Projects」→「+ New project」
  2. プロジェクト名：「見積書自動入力」
  3. 「Instructions」欄に、別ファイルでお渡しした claude_project_prompt.md の内容をそのまま貼り付け
  4. 保存したら、そのプロジェクト内で URL（例：https://example.com）だけ送信
  5. 返ってきた JSON ブロック全体をコピー → ここに貼り付け → 「適用」
